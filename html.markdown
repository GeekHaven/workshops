# your first web page

Save this text as a file:

```
<b>whatever</b>
```

give it a `.html` extension and open your file in a web browser.

---

# more tags:

* `<i>...</i>` - make text italicized
* `<u>...</u>` - underline some text
* `<img src="whatever.png">` - show an image!
